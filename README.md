## Getting Started
- Use this template for starter with golang net/http library
- Or fork this repository

## Requirement
- Go 1.22 or later
- Docker

## Feature
- logging with zap
- recovery
- access log
- gracefully shutdown

### Build and run the server
```sh
$ make dev
```
- this will build the server and run it on port 80
- Checkout Makefile for modifying
